# Carpeta de iconos de Electron

Coloca aquí los siguientes archivos de iconos:

- icon.png (512x512 px) - Icono principal
- icon.ico (para Windows)
- icon.icns (para macOS)  
- tray-icon.png (48x48 px) - Icono para la bandeja del sistema

Puedes generar los iconos en:
- https://www.electron.build/icons
- https://icon.kitchen/
- https://www.iconfinder.com/

Por ahora, la aplicación usará un icono por defecto si no encuentra estos archivos.
